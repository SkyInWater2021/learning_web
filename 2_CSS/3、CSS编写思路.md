## 二、CSS编写顺序的思路

- 先确定盒子的布局
  - position: absolute
  - float: left/right
  - display: flex
- 盒子的特性和可见性
  - display: block/inline-block/inline/none
  - visibility/opacity
- 盒子模型
  - width/height
  - box-sizing
  - margin/border/padding/content
  - box-shadow/text-shadow
- 内部的文本文字
  - font/text
- background
  - background-image/size/position/color
- 其他
  - transform/transition/overflow/white-space
  - ...
