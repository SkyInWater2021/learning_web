# Emmit 语法



## 一、HTML 中的 Emit语法

- ! 或 html:5可以快速生成完整结构的html5代码

- 子代 > 、兄弟 +

- 多个 * 、 上一级 ^

-  分组 ( )、内容 { } 、 数字 $



## 二、 CSS 中的 Emmit 语法

-  w100 -> width:100px

-  h30+m40+p50 -> height:30px;margin40px;padding:50px

- 20-30-40-50 -> margin:20px 30px 40px 50px

-  dib -> display:inline-block
